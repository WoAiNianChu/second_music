# second_music
一个聚合音乐客户端。

## 支持平台
- [x] 网易云音乐
- [x] QQ音乐
- [x] 虾米音乐
- [ ] 酷狗音乐
- [ ] 酷我音乐
- [ ] 哔哩哔哩音乐

## 已完成功能
- [x] 热门推荐
- [x] 搜索单曲、歌单、歌手、专辑
- [x] 本地收藏管理

## 待完成功能
- [ ] 后台播放
- [ ] 本地音乐
- [ ] 播放历史
- [ ] 歌词
